# designers_app_ui

Random Designers Flutter Application with 2 functional screens.

- Dashboard Screen
- Home Screen

## Features/Implementations

- Material 3
- Gradient Appbar
- Profile Avatar
- Route Navigation
- Vertical ListView
- Horizontal ListView
- Custom Gradient Painters
- Gradient Tabs
- Custom Font
- Animation
- Etc.,
